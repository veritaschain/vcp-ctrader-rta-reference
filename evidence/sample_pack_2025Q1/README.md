## Verification Details

The verification script performs:

1. **Event Hash Verification**: Recomputes `SHA256(Canonical(Header) || Canonical(Payload))` for each event
2. **Hash Chain Verification**: Confirms each event's `PrevHash` matches the previous event's `EventHash`
3. **Merkle Root Reconstruction**: Rebuilds the entire Merkle tree from 21 event hashes
4. **Inclusion Proof Verification**: Validates audit paths using RFC 6962 domain separation
5. **Anchor Verification**: Confirms anchor proof integrity

## Third-Party Verification

Independent auditors can verify this pack by:

1. **Recompute all event hashes** using VCP v1.1 hash formula
2. **Rebuild the Merkle tree** from event hashes
3. **Verify inclusion proofs** using RFC 6962 algorithm
4. **Compare anchor timestamps** with external references

```python
# Example: Verify single event hash
import json, hashlib

def canonicalize(obj):
    return json.dumps(obj, sort_keys=True, separators=(',', ':'))

def compute_event_hash(event):
    header = {k: v for k, v in event['Header'].items() if k != 'EventHash'}
    payload = {k: v for k, v in event.items() if k != 'Header'}
    return hashlib.sha256((canonicalize(header) + canonicalize(payload)).encode()).hexdigest()

# Load and verify
with open('events.json') as f:
    events = json.load(f)['events']

for e in events:
    computed = compute_event_hash(e)
    stored = e['Header']['EventHash']
    assert computed == stored, f"Hash mismatch for {e['Header']['EventID']}"

print("All hashes verified!")
```
